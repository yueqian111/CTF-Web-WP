## 代码审计

靶场提供的核心代码如下：

```PHP

<?php
$num = $_GET['num'];
if (!is_numeric($num)) {
    echo $num;
    if ($num == 1) {
        echo 'flag{**********}';
    }
}
?>
```

### 关键逻辑分析

1. **`is_numeric($num)`**：该函数用于检测变量是否为数字或数字字符串。如果是，返回 `true`；否则返回 `false`。

    - 代码中使用了 `!is_numeric($num)`，意味着只有当 `$num` **不是**数字或数字字符串时，才会进入内层判断。

2. **`$num == 1`**：这里使用了 PHP 的弱类型比较运算符 `==`。在弱比较中，PHP 会自动进行类型转换：

    - 当字符串与数字比较时，PHP 会尝试将字符串转换为数字。例如，字符串 `'1a'` 会被转换为数字 `1`。

## 解题思路

要成功获取 flag，需要同时满足以下两个条件：

1. `!is_numeric($num)` → `$num` 不能是纯数字或数字字符串。

2. `$num == 1` → 在弱类型比较下，`$num` 与数字 `1` 相等。

### Payload 构造

我们可以构造一个以数字 `1` 开头，但后面跟有非数字字符的字符串，例如 `1a`：

- `is_numeric('1a')` → 返回 `false`，满足第一个条件。

- `'1a' == 1` → 弱比较时，`'1a'` 被转换为 `1`，满足第二个条件。
