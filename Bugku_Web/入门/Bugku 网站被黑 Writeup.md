## 二、信息收集：目录扫描（dirsearch）

```Bash

dirsearch -u http://117.72.52.127:11647/ -e php,html,js,txt
```

扫描结果中发现关键文件：

```Plain Text

```

该文件为WebShell后门，是突破点。

3. 使用Burp Suite抓包，拦截登录请求，发现请求为POST方式，参数为`pass=1`，目标地址为`/shell.php`。

## 四、密码爆破（Burp Suite Intruder）

1. 将拦截的POST请求发送到Burp Suite的`Intruder`模块。

2. 在`Positions`标签页，选中`pass`参数的值（`1`），设置为爆破位置（§1§）。

3. 在`Payloads`标签页，加载弱密码字典（如`中国网民-弱密码字典T1000.txt`）作为攻击载荷。

4. 开始爆破，观察响应结果：

    - 大部分payload对应的响应长度为`1200`，提示密码错误。

    - 当payload为`hack`时，响应长度变为`1203`，说明密码验证通过。
