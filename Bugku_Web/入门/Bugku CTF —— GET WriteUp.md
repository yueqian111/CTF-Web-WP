## 解题思路

1. **代码审计**：访问靶场后，页面直接显示了后端PHP代码：

    ```PHP
    
    $what = $_GET['what'];
    echo $what;
    if ($what == 'flag') {
        echo 'flag{****}';
    }
    ```

    代码逻辑清晰：

    - 接收一个名为`what`的GET参数。

    - 直接输出该参数的值。

    - 当`what`的值严格等于字符串`'flag'`时，输出`flag{****}`，其中`****`即为真实的flag。

2. **构造请求**：根据代码逻辑，我们只需要通过GET方式传递`what=flag`，即可触发条件，获取到真实的flag。
