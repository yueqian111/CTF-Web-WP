# 前端限制绕过与代码注入：Web 题解题步骤梳理

这道题是典型的**前端限制绕过 + 代码注入**类WEB题，我帮你把完整的解题步骤梳理一下：

---

### 1. 解除输入框长度限制

你已经发现了关键问题：输入框的`maxlength="1"`限制了只能输入一位数。

- 按F12打开开发者工具，在`Elements`面板找到对应的`<input>`标签。

- 把`maxlength="1"`修改为`maxlength="100"`（或直接删除这个属性），这样就能输入任意长度的字符串了。

---

### 2. 利用代码执行漏洞获取flag

这类“计算器”题目，后端通常会直接执行用户输入的内容，存在严重的**代码注入**漏洞。你可以构造Payload来读取flag：

#### 常见Payload（PHP环境）：

- 执行系统命令读取flag：

    ```PHP
    
    system('cat /flag')
    ```

- 或者直接读取文件：

    ```PHP
    
    echo file_get_contents('/flag')
    ```

把构造好的Payload输入到修改后的输入框中，点击“计算”，后端执行后就会返回flag内容。（直接输入正确答案，也可以得到flag）
